MangBeli App

Project Information

  • Category : Android (Kotlin)
  • Find more on :  
  • Overview : Mang Beli is a mobile application designed to connect users with street vendors in their vicinity, making it easier to locate and interact with them. The app aims to save users time and effort by providing a platform where they can find nearby vendors, view detailed information, and get directions to reach them.
  • Objectives:
    • Allow user registration and login authentication via a secure custom API.
    • Enable users to locate and view a list of nearby mobile vendors.
    • Integrate Google Maps API to display vendor locations and provide navigation routes.
    • Provide users with detailed information about each vendor, including location and estimated travel time.
    • Ensure smooth and user-friendly interface for a seamless user experience
