My Anime List

Project Information

  • Category : Android (Kotlin)
  • Find more on :  
  • Overview: Welcome to MyAnimeList! This is my fourth Android application project, developed to provide a comprehensive list of anime and details about each one. The application was completed as part of a certification program on Dicoding, and the certification can be viewed here. This project build by Jetpack Compose.
  • Objectives:
    • Fetch MyAnimeList API and display a list of Anime
    • Fetch detailed information about the chosen anime from the MyAnimeList API.
    • Implement a search feature to find anime by title
    • Allow users to mark and manage favorite anime
